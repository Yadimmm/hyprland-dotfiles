# Dotfiles – Hyprland Cyberpunk Setup

## Sistema
- OS: EndeavourOS (Arch)
- WM: Hyprland
- Bar: Waybar
- Widgets / Powermenu: Eww (Wayland)
- Launcher: Wofi
- Resolución: 2560x1600 @120Hz
- Scale: 1.33

## Eww Powermenu
- Gradiente animado en movimiento
- Glow pulsante
- Click fuera para cerrar
- SUPER+Q para cerrar
- Wayland native

## Waybar
- Botón de apagado custom
- Estilo cyberpunk
- Modular

## Hyprland
- Gaps y bordes neon
- Binds personalizados
- Optimizado para HiDPI

## Atajos importantes
- SUPER + Q → cerrar powermenu
- SUPER + W → cerrar ventana

## Notas
- Los archivos en ~/.config están enlazados a ~/dotfiles
- Editar siempre desde VS Code
